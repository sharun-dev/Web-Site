# TalentTrade 🚀

Welcome to **TalentTrade**, a modern peer-to-peer marketplace web application designed for independent creators, tech developers, and digital artists. The platform allows users to showcase, buy, sell, or rent digital assets—including software applications, web models, creative photos, handmade designs, and tech gear.

---

## 🌟 Features

*   **Dual Transaction Model:** Supports listing items for both direct **Sale** and temporary **Rent**.
*   **Dynamic Media Hosting:** High-performance image and preview asset rendering powered by Cloudinary.
*   **Real-Time Data Layer:** Scalable user accounts, item collections, and user reviews driven by Firebase Firestore.
*   **Responsive UI:** A clean, modern, fully responsive dashboard built entirely with semantic HTML5 and custom CSS3.
*   **Interactive Reviews & Ratings:** Transparent feedback system for buyers and sellers to rate their transaction experiences.

---

## 🛠️ Tech Stack

*   **Frontend:** HTML5, CSS3 (Custom Grid/Flexbox), JavaScript (Modern ES6+ Modules)
*   **Database & Backend Service:** Firebase Firestore
*   **Media Storage Management:** Cloudinary API
